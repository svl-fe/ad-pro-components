# pro-components

SVL FE Business Components

> 业务组件库包含了大量的第三方组件库

## 一、项目启动

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father-build`,

```bash
$ npm run build
```
